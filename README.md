# valorant-friends-deleter
名前の通りvalorantのフレンドを自動削除するツールです。
フレンド用に作成したのでアップデートや英語対応は行わないつもりです。
1920x1080のみ
## 使用方法
### Python3をダウンロード
Python3をPCにダウンロードしてください。
### setup.batを実行
実行してしばらくするとロゴが出るはずです。出ない場合はPythonの環境を再確認してみてください
### valorantを実行
右側のフレンド欄を画像の状態にし、ウィンドウフルスクリーンにしてください。
